### **¿Qué es Java?**

Java es un lenguaje de programación muy popular que se usa para crear aplicaciones de todo tipo: desde programas de escritorio hasta páginas web y apps móviles.

🔑 **Características principales**
- **Orientado a objetos:** organiza el código en “objetos” (como si fueran cosas del mundo real).
- **Multiplataforma:** puedes escribir el programa una vez y ejecutarlo en diferentes sistemas (Windows, Linux, Mac) gracias a la máquina virtual de Java (JVM).
- **Fuertemente tipado:** debes declarar el tipo de datos.
- **Seguro y robusto:** maneja errores y protege la memoria.
